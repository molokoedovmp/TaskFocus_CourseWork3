##Курсовая работа 3 курс "Трекер задач"

### Установить пакеты

```shell
npm i
```

### Настроить .env file


```js
# Deployment used by `npx convex dev`
CONVEX_DEPLOYMENT=
NEXT_PUBLIC_CONVEX_URL=

NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=
CLERK_SECRET_KEY=

EDGE_STORE_ACCESS_KEY=
EDGE_STORE_SECRET_KEY=
```

### Ввести в терминал

```shell
npx convex dev

```

### Запустить приложение

```shell
npm run dev
```
